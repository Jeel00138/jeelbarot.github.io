## Quick Start

### Viewing Locally

1. Navigate to the portfolio directory:
```bash
cd /app/portfolio
```

2. Open `index.html` in your browser or serve via any HTTP server:
```bash
python3 -m http.server 8080
```

3. Visit: `http://localhost:8080`

### Access via Main App

The portfolio is integrated into the main application at:
- **URL**: `http://localhost:3000/index.html`

## Customization

### Updating Content

1. **Personal Info**: Edit the text content in `index.html`
2. **Colors**: Modify CSS variables in `:root` selector in `styles.css`
3. **Fonts**: Change Google Fonts import links in `index.html`
4. **Resume**: Replace `resume.pdf` with actual resume file

### Color Scheme

```css
--color-primary: #3B82F6;        /* Soft Blue */
--color-primary-hover: #2563EB;   /* Darker Blue */
--color-background: #F8FAFC;      /* Light Gray */
--color-surface: #FFFFFF;         /* White */
```
